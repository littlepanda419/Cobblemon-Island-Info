Cobblemon Island 伺服器指令介紹

一般
指令 描述 備註
/rewards 獎勵系統 目前維護中
/msg [username] [text] 私訊玩家訊息 -
/reply 快速回覆上面私訊 也可以點擊玩家ID回覆
/kits 領取套組獎勵 每周刷新 可領取claimtool和pokedex
/vote 投票 每天or根據網頁規則刷新
可以直接開啟全網頁加入書籤較方便
/spawn 回到出生地 同/server hub
Claim相關
指令 描述 備註
/trust [username] [role] 站在領地上信任別人
給予別人此地相關權限		 [role]: Accessor, Container, Builder, Manager
由左至右分別可以基本進入>開箱子/種地/動物>建造破壞>殺寶可夢(全部權限)
/claiminspect 檢視腳下土地為誰所有 -
/claimtool 劃定領地 10分鐘+10個 claim block
tp相關
指令 描述 備註
/home [homename] tp回家 需先 /sethome 或是可用 /homes 選擇 初始僅有2個home可使用
/tpa [username] tp到該玩家處，需經過他同意 被tp者可以打/tpaccept 或點擊聊天框的accept同意請求
/back tp 回上次tp來/死亡點 可以走到點之後 /home回家/放東西 再用/back快速回去冒險
/warps 開啟ui 可快速傳送 -
/server [place] 立刻傳送至該地點/分流 [hub, sunrealm, moonrealm, starrealm, nether, end, eventrealm]
/rtp 隨機tp至任意地點 冷卻3分鐘
寶可夢相關
指令 描述 備註
/hunts 開啟hunts ui 可以狩獵寶可夢 難度簡單到難分別為
指定寶可夢+指定球/+指定性別/+指定性格/+指定特性
/pokefishing 釣魚系統 後面可加 list 目前似乎disable中
/wondertrade 神奇交換
隨機交換別人丟上去的寶可夢一隻		 只能在spawn使用 冷卻X分鐘(待確認)
/daycare 開啟介面 讓兩隻寶可夢瘋狂ii -
/pokegift [1-6] [username] 把第[1-6]的欄位的寶可夢
送給[username]		 同realm才可work
/pokebuilder 寶可夢精進系統
可以改IV、EV、性別、球種、閃光、許多設定 		只可以改變非神獸 只能在spawn使用
/lastlegend 查詢最近出生5隻神獸 出生時間一律是25分左右 可以直接左邊選單去神獸重生計時頁面
交易相關
指令 描述 備註
/bal 查詢pokedollar -
/pay [username] [amount] 給[username] [amount]塊pokedollar /pay Littlepanda0419 20000
/shop 官方遊戲幣商店 -
/store 官方課金商店 store.cobblemonislands.com
/gts 玩家交易 可以交易道具和寶可夢 -
/gts add pokemon [1~6] [總價] 在gts上架 寶可夢須在左邊 -
/gts add item [1~9] [總價] 在gts上架 物品須在快捷欄 -
